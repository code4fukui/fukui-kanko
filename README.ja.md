# みんなでつくる福井の観光アプリ

観光アプリを誰でも簡単に作れる Webツールを提供しています。福井の魅力を発信しましょう。

## デモ
[みんなの福井の観光アプリ - Issues](https://github.com/code4fukui/fukui-kanko/issues)にアプリのURLが掲載されています。皆さんの作品をぜひチェックしてみてください。

## 機能
- Webブラウザ上で簡単に観光アプリを作成できる
- 自分のGitHubアカウントでホスティングできるので公開が簡単
- 地図や写真検索など、便利なツールも用意

## 使い方
1. [ES-Jam](https://code4fukui.github.io/htmlprac/)を開き、サンプルプログラムを元に作成する
2. 自分のGitHubアカウントで新しいリポジトリを作成
3. プログラムをリポジトリにアップロードし、GitHub Pagesで公開
